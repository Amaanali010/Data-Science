# 📊 Descriptive Statistics

## 1. Definition

**Descriptive Statistics** is a branch of statistics that focuses on **summarizing, organizing, and presenting data** in a meaningful way.

---

## 2. Simple Explanation

Descriptive statistics helps you **describe the main features of a dataset** using numbers, tables, and graphs.

---

## 3. In Easy Words

Instead of looking at a long list of data, descriptive statistics **turns it into simple information** that is easy to understand.

---

## 4. Main Components

### 4.1 Measures of Central Tendency (Center of Data)

These help us find the **central or typical value**.

#### 🔹 Example Data:

```
60, 70, 80, 90, 100
```

* **Mean (Average)**
  = (60 + 70 + 80 + 90 + 100) ÷ 5 = **80**

* **Median (Middle Value)**
  = **80**

* **Mode (Most Frequent Value)**
  = No mode (all values are different)

---

### 4.2 Measures of Dispersion (Spread of Data)

* **Range** → Difference between highest and lowest
* **Variance** → How data varies from the mean
* **Standard Deviation** → How spread out the data is

---

### 4.3 Data Presentation

* Tables
* Charts (bar chart, pie chart)
* Graphs (histogram, line graph)

---

## 5. Example

Suppose you have marks of students:

```
60, 70, 80, 90, 100
```

Descriptive statistics helps to find:

* Mean = 80
* Highest value = 100
* Lowest value = 60
* Spread of data

---

## 6. Key Points

* Descriptive statistics **summarizes data**
* It makes data **easy to understand**
* It does **not make predictions**
* It only describes the data you already have
